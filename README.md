## Kali linux/Ubuntu etc. Installation
```
sudo su
apt install git python3 -y
python3 -m pip install pillow requests pytesseract
git clone https://github.com/7wp81x/Zytra
cd Zytra
```

## Termux Installation
```
pkg install git -y
git clone https://github.com/7wp81x/Zytra
cd Zytra
bash install.sh
```

## Usage:
```
python3 zytra.py -h
```
